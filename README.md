## Installation

-   `git clone https://github.com/itminanbiva078/sunShopEcommerce.git
-   `cd SunShopEcommerce/`
-   `composer install`
-   `cp .env.example .env`
-   Update `.env` and set your database credentials
-   `php artisan key:generate`
-   `php artisan migrate`
-   `php artisan db:seed`
-   `npm install`
-   `npm run dev`
-   `php artisan serve`
